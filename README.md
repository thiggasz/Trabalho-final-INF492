# Trabalho final - INF 492

Repositório para acompanhamento do trabalho fianl de disciplina de Redes Neurais Convolucionais. O trabalho aborda o problema de classificação de lesões de pele em benignas ou malignas, utilizando para isso o conjunto de dados ISIC 2024 e as arquiteturas Resnet50 e EfficientNet-B1.
